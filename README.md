# Introducción a la Ciberseguridad

Repositorio de apuntes, resúmenes y laboratorios prácticos desarrollados para la materia **Introducción a la Ciberseguridad**.

## Contenido

### 01. Fundamentos

* ¿Qué es la Ciberseguridad?
* Tríada CIA
* Riesgos, Amenazas y Vulnerabilidades

### 02. Sistemas y Criptografía

* Sistemas Numéricos (Decimal, Binario y Hexadecimal)
* Conversiones entre Sistemas Numéricos
* ASCII y Base64
* Introducción a la Criptografía
* Criptografía Simétrica
* Criptografía Asimétrica
* XOR

### 03. Redes

* Direcciones IP
* Máscara de Subred
* NAT
* NIC (Network Interface Card)

### 04. Ataques

* Malware
* Phishing
* Fuerza Bruta
* Ransomware

### 05. Defensa

* IDS, HIDS y NIDS
* Mitigación de Riesgos
* Contraseñas Seguras

### 06. Laboratorios

* Laboratorio XOR
* Laboratorio GoPhish
* Laboratorio Ransomware
* Laboratorio HTML Injection

---

## Objetivos

* Comprender los fundamentos de la ciberseguridad.
* Identificar amenazas y vulnerabilidades comunes.
* Conocer mecanismos básicos de protección.
* Entender conceptos fundamentales de criptografía y redes.
* Introducir prácticas básicas de análisis y defensa.
